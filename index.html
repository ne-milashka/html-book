Центрирование блока относительно родителя

Этот фокус позволяет центрировать блок заранее определенной ширины независимо от ширины экрана. Кроме того, если использовать max-width вместо width, то, если размер экрана пользователя окажется меньше ширины этого блока, то блок не выйдет за границы экрана, а будет равен ширине экрана. Это весьма распространенная практика

Пример:

Код:


    



            
.example1-container {
    padding: 10px;
    background-color: lightblue;
}

.example1-content {
    height: 100px;
    max-width: 500px;
    background-color: teal;
    margin: 0 auto; /* центрирует блок так, что правый и левый маргин становятся одинаковыми */
}

Вертикальное и горизонтальное центрирование блока относительно релятивного контейнера

Эта техника применяется, в основном, для вёрстки модальных окон

Пример:

Код:

    

        

    

                
    .example2-container {
        position: relative;
        width: 100%;
        height: 300px;
        background: lightblue;
    }
    
    .example2-block {
        position: absolute; /* позволяет двигать блок относительно родительского релятивного контейнера */
        top: 50%;           /* смещаем блок на 50% от высоты родителя */
        left: 50%;          /* смещаем блок на 50% от ширины родителя */
        transform: translate(-50%, -50%); /* смещаем блок на 50% от ширины и высоты этого блока */
        width: 120px;
        height: 80px;
        background: teal;
    }
